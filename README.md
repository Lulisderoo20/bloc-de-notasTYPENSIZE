# Bloc de notas NTS

Replica simple de Bloc de notas hecha en HTML, CSS y JavaScript.

## Que hace ahora

- Interfaz inspirada en Notepad de Windows
- Menus de Archivo, Editar, Formato, Ver y Ayuda
- Edicion de texto en la pantalla principal
- Cambio de fuente y tamano solo sobre la seleccion
- Ajuste de linea
- Estado con linea, columna, cantidad de caracteres y lineas
- Atajos `Ctrl+N`, `Ctrl+O`, `Ctrl+S` y `Ctrl+A`

## Guardado

- Si el documento no tiene formato especial, se guarda como `.txt`
- Si usaste fuentes o tamanos distintos dentro del texto, se guarda como `.html` para conservar el formato

## Como abrirlo

1. Abri `index.html` en tu navegador.
2. Escribi o pega texto en el editor.
3. Selecciona una parte del texto.
4. Cambia la fuente o el tamano desde la barra superior.
